# ☕ QuanLyCoffe

## 📌 Giới thiệu
Phần mềm quản lý quán cafe viết bằng WinForms.

## 🚀 Công nghệ
- .NET 8
- Entity Framework Core
- SQL Server

## Buổi 2
- Tạo các form nhân viên, bàn, phân công nhân viên, lịch làm

## Các lỗi gặp phải
- Bàn chưa thể hiện được bàn trống và bàn có khách vẫn còn ở dạng chuỗi, làm người dùng khó phân biệt 
- lịch làm chưa hiển thị được lịch do chưa tạo bảng lịch làm

## Cách khắc phục
- Sửa lại kiểu dữ liêuj cho trạng thái bàn, thay textbox bằng checkbox
- Tạo bảng lịch làm chạy lại cơ sở dữ liệu
